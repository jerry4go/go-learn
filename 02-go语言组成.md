# Go 语言基础语法

字符串连接

```golang
package main
import "fmt"
func main() {
    fmt.Println("Google" + "Runoob")
}
```

Go 语言变量

声明变量的一般形式是使用 var 关键字：

var identifier type

可以一次声明多个变量：

var identifier1, identifier2 type

```golang
package main
import "fmt"
func main() {
    var a string = "Runoob"
    fmt.Println(a)

    var b, c int = 1, 2
    fmt.Println(b, c)
}
```
